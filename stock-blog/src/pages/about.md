当サイトは、米国の代表的な株価指数 **S&P 500・S&P 400・S&P 600**（大型・中型・小型株）に組み入れられている全銘柄のファンダメンタル情報を、日本語で分かりやすく整理することを目指す **個人運営の情報サイト** です。

## サイトの目的

- 米国株投資を始めた、または始めようとしている **日本の個人投資家** が、企業の財務状況や評価指標を **日本語のまま** スピーディに把握できる場を提供すること。
- 各銘柄の財務指標（PER、PBR、ROE、配当利回り、売上・利益成長率、負債比率など）を **同じ尺度・同じ見せ方** で比較しやすく表示すること。
- アナリスト評価や DCF（割引キャッシュフロー）に基づく理論株価の目安を併記し、利用者ご自身の判断材料を増やすこと。
- 主要な金融指標について、**初心者でも理解できる平易な解説** を併設すること。

当サイトは特定の銘柄を推奨することを目的とはしておらず、利用者の方が自分自身で考え、判断する材料を提供する立場をとります。

## 運営者について

- **運営者名**：アメリカ株インサイト 管理人
- **運営形態**：個人運営
- **運営者のスタンス**：金融機関に所属する登録外務員ではなく、特定の証券会社等から報酬を得て情報を恣意的に発信する立場ではありません。米国株を保有する一個人投資家として、自分自身が「あったら便利だ」と思うデータの整理・可視化を行っています。
- **連絡先**：[お問い合わせ](/contact)ページよりご連絡ください。

## データソース

| 項目 | データソース | 備考 |
|------|-------------|------|
| 株価・財務データ | Yahoo Finance（yfinance ライブラリ経由）| 自動取得、日次更新 |
| 構成銘柄一覧 | S&P 500 / 400 / 600 構成銘柄リスト | Wikipedia 等の公開情報をベースに整備 |
| アナリスト評価 | Yahoo Finance 集計値 | 取得時点のスナップショット |
| チャート・株価ウィジェット | TradingView 公式埋め込み | 同社のサービスを利用 |
| 企業概要・要約の日本語化 | 公開情報をもとに自動翻訳・要約 | 必要に応じて編集を加えています |

データはあくまで自動取得・自動加工によるものであり、最終的な投資判断にあたっては各企業の **公式 IR・SEC 提出書類** をご確認いただくようお願いいたします。詳しくは [免責事項](/disclaimer) をご覧ください。

## 更新頻度

- **株価・指標**：米国市場の取引終了後、日次で自動更新（営業日ベース）
- **銘柄構成**：S&P 500・400・600 の構成銘柄に変更があった際に随時反映
- **解説記事・コンテンツ**：随時拡充

## 表示している主な指標について

各銘柄ページには、投資判断の参考となる以下のような指標を掲載しています。各指標の意味は、用語解説をご参照ください。

- **PER（株価収益率）** … 株価が 1 株あたり利益の何倍かを示す指標 → [PER とは](/posts/per-explained)
- **ROE（自己資本利益率）** … 株主資本をどれだけ効率的に利益に変えているか → [ROE とは](/posts/roe-explained)
- **配当利回り** … 株価に対して 1 年間に支払われる配当金の割合
- **売上・利益成長率（YoY）** … 前年同期比の伸び率
- **負債比率（D/E）** … 自己資本に対する負債の比率
- **DCF 理論株価** … 将来キャッシュフローの現在価値合計から推計した理論的な株価 → [DCF とは](/posts/dcf-explained)

## サイトの構成

- **トップページ** … セクター別の銘柄一覧、株価変動率
- **銘柄詳細ページ** … 株価チャート、財務ハイライト、業績推移、アナリスト評価、競合銘柄、日本の証券会社での取扱状況
- **A-Z 一覧** … 全銘柄をアルファベット順に確認できるインデックス
- **解説記事** … 投資指標や用語の入門解説
- **検索** … 銘柄シンボル・社名・セクター等から横断検索

## 当サイトの特徴

- **一覧性と比較容易性**：S&P 500・400・600 の全銘柄を共通フォーマットで表示し、横断的な比較が可能。
- **日本語ファースト**：英語の財務情報を日本語に置き換え、日本の投資家がストレスなく読めるように整備。
- **証券会社別取扱情報**：日本の主要ネット証券（Monex、SBI、楽天等）での取扱状況をまとめて参照可能。
- **継続的なデータ品質改善**：誤訳・誤データに気づいた場合は [お問い合わせ](/contact) よりご報告いただけます。

## 投資判断の前に必ずお読みください

当サイトに掲載される情報は、特定の銘柄や金融商品の購入・売却を推奨するものではありません。投資はリスクを伴うものであり、最終的な判断は必ずご自身で行ってください。詳細は [免責事項](/disclaimer) を必ずご覧ください。
